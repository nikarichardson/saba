Virtualization & Concurrency

Important Concepts for OS

Virtualization is the process of taking a physical resource and making it virtual in order to duplicate the resource. This process allows multiple programs to run by sharing the CPU. Thus, the memory is effectively private to each running program, not a communal resource. Virtualization duplicates the CPU as many times as necessary for the operating system.
